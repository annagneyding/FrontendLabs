# FrontendLabs
Лабораторная работа №1, в ходе которой было создано приложение "Список дел" и реализован следующий функционал:

-каркас сайта при помощи языка разметки html
-добавление, изменение, сохранение задач при помощи языка JavaScript 
-фильтрация списка дел по принципу: все задачи, сделанные, не сделанные
-при помощи css страница была оформлена определёнными стилями
